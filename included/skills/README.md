# Included Skills

This directory contains one physical mirror for each of the `673` cataloged source-backed skills.

Skill mirrors are grouped by category and source tier, then isolated in a stable repository-and-path `install_name` directory so duplicate upstream skill names do not overwrite each other. Use `manifest.json` to trace every mirror back to its immutable source URL, commit, source path, and SHA-256 hashes.

Do not bulk-install entries where `bulk_install_safe` is false.
