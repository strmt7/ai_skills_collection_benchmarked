# Included Skills

This directory contains one physical mirror for each of the `673` cataloged source-backed skills.

Skill mirrors are grouped by category and source tier, then isolated in a stable `install_name` directory. The install name is the skill name when unique, with exact source-derived qualifiers only when needed to prevent collisions. Use `manifest.json` to trace every mirror back to its full immutable source URL, commit, source path, and SHA-256 hashes.

Do not bulk-install entries where `bulk_install_safe` is false.
