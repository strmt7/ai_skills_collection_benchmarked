## Brand Distinction Strategy

To make an infographic stand out, **prioritize brand identity over industry norms**.

1. **Avoid the "Tech Blue" Default:** Unless the brand explicitly uses blue, avoid defaulting to it. Blue and purple are the most overused colors in the space.
2. **Logo-First Extraction:** Extract the dominant color from the client's logo and build a complementary or split-complementary palette around it.
3. **The "Non-Generic" Filter:** If the brand lacks specific colors, choose high-contrast, memorable pairings that feel "editorial" rather than "corporate" (e.g., Earthy greens + bright oranges, or deep plums + acid leathers).

## Light Mode Palettes (When background: "light")

Only use when the client explicitly requests light/white output. These are rare in your work (< 1%) but defined here for completeness and client requests.

When `background: "light"` is set, replace all dark tokens with these:

```css
/* Light mode token override — apply to :root */
:root {
  --bg:          #FFFFFF;
  --bg-secondary:#F7F7F8;
  --surface:     #FFFFFF;
  --border:      rgba(0,0,0,0.08);
  --text:        #0D0D0D;
  --muted:       #6B7280;
  --shadow:      0 2px 12px rgba(0,0,0,0.08);
}
/* Cards on light backgrounds get a subtle shadow instead of border glow */
.card {
  border: 1px solid var(--border);
  box-shadow: var(--shadow);
}
/* Glow effects are DISABLED on light backgrounds — they look wrong on white */
```
