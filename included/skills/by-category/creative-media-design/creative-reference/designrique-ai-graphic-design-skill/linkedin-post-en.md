AI doesn't replace designers. It exposes the ones who never learned the fundamentals.

I've spent years crafting brand identities by hand — Bezier curves, color theory, typographic hierarchy. Then I got a Master's in Computer Science and studied how machines learn to see.

When generative AI hit graphic design, I watched two camps form:

Camp 1: "AI will kill design." Refused to touch it.
Camp 2: "AI does it all." Surrendered creative control entirely.

Both are wrong.

Camp 2 produces what the industry now calls "AI slop" — generic logos with pastel gradients, no brand coherence, zero mathematical precision. It looks AI-generated because nobody directed it.

The real shift isn't AI replacing designers.
It's designers becoming Creative Directors of AI.

I needed to scale visual identity production for multiple clients without losing the essence — the intentional color choices, the precise vector geometry, the strategic brand archetypes. So I built a system.

I curated 46 specialized sources, distilled them across 5 research rounds, and compiled everything into a single, open-source Claude Code skill that covers:

- Tool selection (Recraft for SVG, Midjourney for ideation, Firefly for IP-safe work)
- Prompt engineering formulas per tool (not generic — each AI speaks a different language)
- A full vectorization pipeline (upscaling models, Vectorizer.ai, Bezier cleanup via Box Method)
- Consistency techniques (Style IDs, seed control, LoRAs)
- IP safety matrix (which tools are safe for commercial use and which aren't)
- Python automation for batch generation
- The 7 anti-patterns that produce AI slop — and how to avoid every one

This isn't theory. It's a production-tested workflow from someone who does both — designs brands and understands the math behind the models.

The skill is open-source and open for contributions.
If you've found techniques that work, I want them in.

Install it: github.com/designrique/ai-graphic-design-skill
Contribute: fork, test your addition, open a PR. All contributions require review.

The tools change every 6 months. The fundamentals last decades.
Learn both.
